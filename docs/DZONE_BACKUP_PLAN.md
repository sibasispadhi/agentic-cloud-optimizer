# DZone Backup Submission Plan

If InfoQ rejects, submit immediately to DZone using the SAME article.

## DZone Submission Process

**URL:** https://dzone.com/pages/contribute

**Process:**
1. Create account (you already have one: https://dzone.com/users/5245819/sibasispadhi.html)
2. Click "Contribute Article"
3. Submit article (copy from Google Doc)
4. Select zone: "Java" or "Performance"
5. Wait for editorial review (usually 1-2 weeks)
